# Network Lab Starter

Browser-based interactive simulator for CS602 hands-on networking labs.

## Run
Open `index.html` in browser (or use VS Code Live Server).

## Included Concepts
- Node/link topology visualization
- Dijkstra shortest-path routing
- Packet animation along computed path
- Link failure simulation and route recomputation
- Routing table display

## Student Tasks
1. Add custom topology presets.
2. Implement weighted random traffic generation.
3. Add split horizon / poison reverse visualization.
4. Export route tables as JSON.
